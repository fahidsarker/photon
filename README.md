## Compile and Run the project
- create a file at `src/.env.local` with API key from RapidAPI:
  ```env
  RAPID_API_URL=<YOUR URL>
  RAPID_API_HOST=<YOUR HOST>
  RAPID_API_KEY=<YOUR KEY>
  ```
- Run `npm install`
- Run `npm run build`
- Run `npm run start`
